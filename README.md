# Cab-Hailing-Application
This is a very basic project, that deploys the Dijkstra's Algorithm to allocate cabs to customers and is built using C. This project was undertaken prior to acquiring proficiency in data structures, with the primary objective being, to attain a strong command of the C programming language.

# Cab Booking System

## Fixed Cab Database

Contains information about Drivers, Vehicles, Speeds, Default Location, and Ratings.

## Fixed Map Database

Contains information about various locations on the map, distances between locations, and surge times for each route taken by drivers.

## User Input Interface

Switches between various services such as Advanced Booking, Cancellation of Trips, and Normal Booking.

## Booking

Books cabs for users based on the current availability of drivers filtered by the Cab Finder Module.

## Advanced Booking

Reserves cabs for users based on their availability in the requested time period.

## Cancellation

Cancels trips for users with reference to the ID assigned to the trip.

## Dynamic Location Tracker

Keeps track of the current location of drivers and their availability status.

## Advanced Availability Checker

Calculates the availability of a driver based on their current trips and requested time periods.

## Trips Manager

Contains information about ongoing trips in the database.

## Availability Updater

Updates the data of available cabs after a booking or cancellation by a user.

## Cab Finder Wizard

Filters out the best available options for the user according to the preferences given.

## Bill Generator

Generates the bill for the user based on booked or canceled trips.
